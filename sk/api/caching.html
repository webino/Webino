Cache Methods¶

  • $app->getCache()
  • $app->setCache()

$app->getCache()¶

Getting cached value.

$myValue = $app->getCache('myCacheKey');

$app->setCache()¶

Setting value to cache.

$app->setCache('myCacheKey', $myValue);
