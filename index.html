Webino™ is a platform for creating quality and modular web applications.
It’s built on PHP with Zend Framework 2 and other useful open-source packages.

Note

We are still working on the contents of this documentation.

Why Use Webino?¶

  • clean code
  • easy to learn / develop / deploy
  • modular architecture
  • well tested

Who Uses Webino?¶

Nobody is currently using this version of Webino™, it’s a prototype.
